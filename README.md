## 🚀 Featured Projects

### 🥗 [Dietary Maps AI](https://github.com/ambalikajaiswal/dietary-maps-ai) — 🏆 Hackathon Winner
`Next.js 14` `React 18` `TypeScript` `Tailwind CSS` `Zustand` `OpenStreetMap`

A multi-agent AI system using **8 specialized agents** to help users discover restaurants matching complex
dietary needs (halal, vegan, gluten-free, keto) via natural language queries. Includes a transparency
confidence-scoring pipeline that rates each result based on verified OSM dietary tags, evidence strength,
data recency, and community signals.

### 🗣️ [ReviewLens](https://github.com/ambalikajaiswal/reviewlens)
`Python` `Hugging Face Transformers` `PyTorch`

A conversational AI system for multilingual sentiment analysis that auto-detects review language, classifies
intent, and delivers natural-language explanations in the user's native language with low-latency, autonomous
response generation.

### 🔐 [ZeroTrace](https://github.com/ambalikajaiswal/zerotrace)
`Go` `TypeScript` `React` `net/http` `Goroutines` `REST APIs`

A full-stack zero-trust secret-sharing application with a TypeScript/React frontend and a Go backend.
Enforces TTL-based credential expiration, concurrent goroutines, thread-safe in-memory storage, and
least-privilege IAM access with automatic secret destruction.
